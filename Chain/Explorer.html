Blockchain Explorer

July 2022 - September 2022

I created this explorer for a single purpose - to track users who bypass paying royalties on the swap of assets. Simply put, royalties is a source of passive income paid to creators when an asset they created is traded, bought, or sold.

When a user makes a swap using smart contracts, royalties are programmed into the smart contract directly (so royalties get paid). However, when users do a trade peer-to-peer, there are no royalties in the sale since the royalties aren't programmed on the asset level.

My explorer iterates through a specific wallet address and will look for the swap of NFTs with the collection identifier EAPES-8f3c1f. If such transaction is found, it will then check for the payments of liquid assets to or from the address (this procedure mimics the peer-to-peer transactions). Each detail, such as hashes, senders, and recievers is hyperlinked to the offical explorer for verification purposes.

You can view my explorer here. You'll see 4 buttons.

Search Address Button

Insert an ERD address to return any potential peer-to-peer transactions done by this address. For example, try searching this address: erd18wa3vz8r4zye7el55alyf7llt9gjnghlturl2ft28a72x6f0jt6smkra3t

Get The Herotag Button

Herotags are a unique trait to the Elrond Blockchain - you can think of a herotag like a username, so instead of transfering money to a wallet address 20 random characters in length, you can type a simple name. For example, try mikey, which will return this persons wallet address.

Find Owner Button

This will scan the chain and return the wallet address or smart-contract in which an NFT is owned by. For example, try searching this unique identifier: EAPES-8f3c1f-2211.

Latest Transactions Button

This is a fairly large request - to it may take a second to process. This will scan the entire chain and return 500 of the most recent transactions in which an NFT will the identifier EAPES-8f3c1f is used. A transaction that includes no smart-contracts is highlighted in red.
